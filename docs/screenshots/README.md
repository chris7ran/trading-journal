# Screenshots

Drop your app screenshots here so they render in the main `README.md`:

- `dashboard.png` — main dashboard
- `trades.png` — trade log / list
- `stats.png` — analytics screen
- `macro.png` — macro terminal

Tip: capture on the iPhone, keep a consistent width (e.g. 1170 px), and compress
(e.g. https://tinypng.com) to keep the repo light.
